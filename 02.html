TIPOS DE NUMEROS

numeros naturais não nulos: representam todos os números inteiros positivos, os naturais não nulos seriam todos os números inteiros positivos maiores que zero. Ou seja, todos os números que não incluem o zero, como 1, 2, 3, 4, 5, ... até a infinitude.
Existem diferentes tipos de números na matemática, cada um com suas características específicas. Os principais tipos de números são:

Números naturais: São os números inteiros positivos, ou seja, aqueles que começam a partir do zero e seguem em ordem crescente. Exemplos de números naturais são 1, 2, 3, 4, 5, entre outros.

Números inteiros: São os números que incluem tanto os naturais quanto seus respectivos opostos negativos. Ou seja, são os números positivos, negativos e o zero. Exemplos de números inteiros são -3, -2, -1, 0, 1, 2, 3, entre outros.

Números racionais: São números que podem ser expressos na forma de fração, ou seja, a razão entre dois números inteiros. Eles podem ser finitos ou infinitos, mas com uma dízima periódica. Exemplos de números racionais são 1/2, 3/4, -2/5, 0,5, entre outros.

Números irracionais: São números que não podem ser expressos na forma de fração, ou seja, não podem ser escritos como uma razão entre dois números inteiros. Eles possuem uma expansão decimal infinita e não periódica. Exemplos de números irracionais são √2, π (pi), e e (número de Euler).

Números reais: É o conjunto que engloba tanto os números racionais quanto os números irracionais. Ou seja, é o conjunto de todos os números que podem ser representados na reta numérica. Os números reais são usados para representar quantidades contínuas, como medidas e valores decimais.
